About This Demo

MediDesk is an AI-powered front desk assistant for hospitals and clinics. It can be embedded on any website using a simple script tag.

Look for the blue chat button in the bottom-right corner of this page!

How to Embed MediDesk

Add this script tag to your website:

<script src="https://your-domain.com/medidesk.js"></script>

Or with custom configuration:

<script src="https://your-domain.com/medidesk.js" data-position="bottom-right" data-theme="blue" data-hospital-id="your-hospital-id" ></script>
